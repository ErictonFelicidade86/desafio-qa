# Automação de Teste com Cypress

## Sobre

**Foco em Teste Automatizado E2E**

## Stacks
- Cypress
- Javascript
- Yarn
- Node.js
- Git
- GitHub Actions

## Excução do teste

1. Instalar as dependências
```
yarn install
```
ou
```
npm install
```

3. Subir o Cypress UI
```
yarn cypress open
```
ou
```
npx cypress open
```

4. Executar testes em Headless
```
yarn cypress run
```
ou
```
npx cypress run
```

5. Executar Allure Result
```
yarn allure
```
ou
```
npm run alure
```

5. Executar Allure Report
```
yarn allure:gene
```
ou
```
npm run alure:gene
```
